# M&S Traders - Tour & Travel Company Website

Welcome to the M&S Traders website repository! This is a modern, responsive website for a tour and travel company offering exciting packages and destinations worldwide.

## 🌍 Features

### Core Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Hero Section** - Eye-catching landing area with call-to-action
- **Tour Packages** - Showcase of 6 featured travel packages with pricing and details
- **Destination Explorer** - Explore top destinations across Europe, Asia, Americas, and Africa
- **Service Highlights** - 6 key services including flights, hotels, tours, and insurance
- **Customer Testimonials** - 5-star reviews from satisfied travelers
- **Contact Form** - Easy-to-use contact form for inquiries
- **Newsletter Signup** - Subscribe to travel deals and tips
- **Smooth Animations** - Beautiful scroll animations and transitions

### Interactive Elements
- Smooth scrolling navigation
- Mobile hamburger menu
- Package card hover effects
- Form validation
- Ripple effects on buttons
- Scroll animations for elements
- Responsive image handling

## 🚀 Getting Started

Simply open `index.html` in your web browser or use a local server:

```bash
python -m http.server 8000
```
